# Predicting T Cell Receptor Specificity with Deep Learning
The accompanying notebook for my blog post Predicting T Cell Receptor Specificity with Deep Learning:  https://medium.com/@yuan_tian/predicting-t-cell-receptor-specificity-with-deep-learning-12757a899e8b


To run the notebook, you need to have Python 3.6 or later and install Pytorch v1 and the fastai library:
`conda install -c pytorch -c fastai fastai`

To use Pip install, please first install Pytorch following the instructions on:
https://pytorch.org/

and then install fastai by
`pip install fastai`
